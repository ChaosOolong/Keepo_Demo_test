This is a demo project based on **M5Stack**, developed with **Arduino**.

## Overview

This project is a smart fridge hardware demo designed to improve the fridge storage and ingredient management experience.

The demo shows how users can interact with a smart fridge system to manage ingredients, check storage status, and perform basic in/out inventory operations.

## Features

- UI interface demo
- Ingredient inventory management
- Ingredient check-in and check-out workflow
- Basic interaction logic for a smart fridge scenario

## My Contribution

The rest of the project was completed in collaboration with another teammate.

## Development

The project is based on **M5Stack** and should be built and uploaded using **Arduino IDE** or another Arduino-compatible environment.

Before compiling, please make sure the required M5Stack libraries are installed.
